Primary (green) is used for actions and progress. Blue accent adds highlights. Error red is used for delete/warnings. Neutral grays provide a clean background.

Typography

Heading Example (24px)

Body Text Example (16px)

The UI uses a sleek sans-serif font with high contrast against the dark background for clear readability.

Buttons

Primary buttons trigger main actions like "Add Game" or "Save". Secondary buttons handle supporting actions like "Back" or "Cancel".

Form Inputs

Dark inputs with clear contrast, used throughout login and game entry forms.

Progress Bars

Progress bars represent checklist completion. Bright green indicates success and progress at a glance.
